# CUTS

## Not Implemented

- Global multi-slot optimization (Hungarian algorithm)
- Real-time METAR API integration
- UI dashboard
- True multi-objective optimization solver
- Historical analytics storage

---

## Rationale

The system prioritizes deterministic and auditable scheduling for safety-critical dispatch.

LLM-based decision-making was intentionally avoided for core scheduling logic to preserve explainability.

Optimization complexity was bounded to maintain clarity, modularity, and reliability.
